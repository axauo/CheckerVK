# CheckerVK
Чекер аккаунтов ВКонтакте. 

Установка Termux:
pkg update && upgrade
pkg install git
pkg install php

Желательно копировать скрипт во внутреннее хранилище так будет проще добавлять базы и прокси

termux-setup-storage
cd /sdcard/
git clone https://github.com/axauo/CheckerVK.git

Использование:
cd CheckerVK
php checker.php
